Diseño Web

Un diseñador web es alguien creativo y técnicamente meticuloso, y usa ambos atributos para construir o rediseñar sitios web. El diseñador web tiene la capacidad de comprender lo que se necesita para que un sitio web sea funcional y fácil de usar, pero al mismo tiempo lo hace estéticamente atractivo para el usuario.

Desarrollo Web

Un desarrollador web crea y mantiene sitios web con el cliente y el consumidor en mente. ... También están muy familiarizados con muchos programas de software, aplicaciones web y lenguajes de programación web, como el lenguaje de marcado de hipertexto (HTML), JavaScript (JS), Ruby on Rails y C ++.
